## Shadow System 🌑

### Levels
1. **Subtle**: 0 1px 2px rgba(15, 63, 70, 0.04)
2. **Light**: 0 2px 6px rgba(15, 118, 110, 0.06)
3. **Medium**: 0 2px 8px rgba(15, 118, 110, 0.08)
4. **Strong**: 0 4px 12px rgba(15, 118, 110, 0.12)
5. **Heavy**: 0 4px 16px rgba(15, 118, 110, 0.15)
6. **Extra**: 0 8px 24px rgba(15, 118, 110, 0.12)

### Usage
- Cards: Medium shadows
- Hover states: Strong shadows
- Panels: Extra shadows
- Buttons: Strong shadows on hover

## Border Radius System 📐

### Sizes
- **Small**: 8px (standard elements)
- **Medium**: 10px (cards, inputs)
- **Large**: 12px (panels, sections)
- **Pill**: 20px (badges, kickers)
- **Circle**: 50% (icons, avatars)

## Spacing System 📏

### Padding

### Gaps

## Responsive Enhancements 📱

### Breakpoints
- **Desktop** (>1180px): Full 6-column layout
- **Tablet** (980-1180px): 3-column layout
- **Mobile** (<980px): Single column, horizontal metrics

### Adaptations
- Jacket metrics change to horizontal on mobile
- Reduced padding on smaller screens
- Maintained visual appeal across devices
